# Альфа Бизнес AI

Демонстрационный прототип финансового помощника для малого бизнеса. Приложение загружает банковскую выгрузку, показывает ключевые показатели, выявляет риски, строит прогноз выручки и сравнивает демонстрационные тарифы.

> Все тарифы и финансовые условия в прототипе являются демонстрационными и не относятся к действующим продуктам Альфа-Банка.

## Возможности

- дашборд с выручкой, расходами, прибылью и средним чеком;
- аналитика операций с фильтрами и разбивкой по категориям;
- прогноз выручки на три месяца в трех сценариях;
- предупреждения об аномалиях и риске кассового разрыва;
- сравнение демонстрационных тарифов и расчет возможной экономии;
- ИИ-помощник, работающий в mock-режиме без API-ключа.

## Архитектура

`pages` отвечает за Streamlit-интерфейс, `services` содержит бизнес-логику, `repositories` изолирует SQLAlchemy, `schemas` описывает структуры данных, а `utils` содержит загрузку, проверку и форматирование. SQLite создается локально при первом запуске.

## Структура

```text
app.py                  # точка входа Streamlit
config.py               # переменные окружения
database.py, models.py  # база данных и ORM
pages/                  # экраны приложения
services/               # расчеты и сценарии
repositories/           # доступ к данным
schemas/                # dataclass/TypedDict-схемы
utils/                  # загрузка, валидация, форматирование
data/                   # демонстрационные CSV
tests/                  # модульные тесты
```

## Стек

Python 3.12, Streamlit, pandas, NumPy, Plotly, SQLAlchemy, scikit-learn, python-dotenv и pytest.

## Формат CSV

Обязательные колонки: `date`, `amount`, `operation_type`, `description`. Дополнительно поддерживаются `category` и `payment_channel`. Допустимые типы операций: `income`, `expense`, `refund`, `commission`. Дата должна распознаваться pandas, сумма должна быть неотрицательной.

Демонстрационный сценарий использует вымышленный интернет-магазин одежды за шесть месяцев: рост продаж, всплеск рекламы, возвраты, крупная закупка и приближение к лимиту переводов. Реальные банковские данные, действующие тарифы и персональные сведения в репозитории отсутствуют.

## Подключение GigaChat API

ИИ-помощник может объяснять уже рассчитанные показатели через GigaChat. Для этого создайте проект GigaChat API, получите ключ авторизации и создайте локальный файл `.env` на основе `.env.example`:

```env
GIGACHAT_CREDENTIALS=ваш_ключ_авторизации
GIGACHAT_SCOPE=GIGACHAT_API_PERS
GIGACHAT_MODEL=GigaChat-2
GIGACHAT_VERIFY_SSL_CERTS=true
```

Установите зависимости и запустите приложение:

```powershell
pip install -r requirements.txt
streamlit run app.py
```

Для обычного локального запуска используйте PowerShell-скрипт, чтобы закрыть старый Streamlit, очистить его кэш и открыть новую клиентскую сессию:

```powershell
.\run_app.ps1
```

`GIGACHAT_API_PERS` подходит для физического лица, `GIGACHAT_API_B2B` — для ИП и юридических лиц с предоплатой, `GIGACHAT_API_CORP` — для корпоративного доступа с постоплатой. Scope должен соответствовать выданному ключу.

Проверка SSL-сертификатов включена по умолчанию. Если в корпоративной сети требуется цепочка сертификатов, укажите путь в `GIGACHAT_CA_BUNDLE_FILE`. Отключение `GIGACHAT_VERIFY_SSL_CERTS=false` допустимо только для локальной разработки и отображается в интерфейсе предупреждением.

Без credentials помощник продолжает работать в демонстрационном режиме и не выполняет сетевых запросов. В GigaChat передаются только агрегированные и обезличенные показатели, предупреждения, прогноз и рекомендация по тарифу. Исходная банковская таблица, реквизиты, персональные данные, credentials и access token не отправляются и не логируются.

Не добавляйте `.env` в Git: он уже указан в `.gitignore`. Ответы помощника не заменяют профессиональную консультацию, прогноз не является гарантией, а тариф никогда не меняется автоматически.

## Ограничения прототипа

Прогноз основан на простой линейной модели и не является финансовой гарантией. Рекомендации помощника не заменяют бухгалтерскую, налоговую или юридическую консультацию. Без `GIGACHAT_CREDENTIALS` ИИ-помощник работает с локальным демонстрационным ответом из рассчитанного контекста.
